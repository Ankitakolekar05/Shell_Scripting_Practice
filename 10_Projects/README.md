# 📖 Project Details

## 📁 Folder Organizer

A Bash script that automatically organizes files into folders according to their extensions.

### Features

- Organizes Images
- Organizes PDF files
- Organizes Music
- Organizes Videos
- Organizes Text files
- Organizes Code files

### Linux Commands Used

- mkdir
- mv
- basename
- case
- for loop

---

## 💾 Backup Script

A Bash automation script that creates backups of a source directory.

### Features

- Creates backup folders automatically
- Uses timestamps
- Copies files and folders
- Compresses backups (tar.gz)

### Linux Commands Used

- cp
- mkdir
- tar
- date

---

## 📊 Log Analyzer

A Bash script that analyzes Linux log files and generates a summary report.

### Features

- Counts INFO messages
- Counts WARNING messages
- Counts ERROR messages
- Generates summary report
- Uses variables and command substitution

### Linux Commands Used

- grep
- wc
- cat
- echo

# 🚀 How to Run

Clone the repository:

```bash
git clone git@github.com:Ankitakolekar05/Shell_Scripting_Practice.git
```

Go to the repository:

```bash
cd Shell_Scripting_Practice
```

Make any script executable:

```bash
chmod +x script_name.sh
```

Run the script:

```bash
./script_name.sh
```
